Сайт: https://lacoste.ru
1. Регистрация нового пользователя без указания пола
	Шаги:
		Страница https://lacoste.ru/account/register/-> 
		Заполнить поля Фамилия, Имя, Телефон, Email-> 
		Отметить поле для согласия с публичной офертой ->
		Нажать на кнопку "Зарегистрироваться"
	Результат: Профиль не будет создан, пользователя известят, о необходимом заполнении поля


2. Добавление товара в корзину
	Шаги:
		Страница https://lacoste.ru/catalog/krossovki-obuv-muzhchiny/ ->
		Выбрать товар: CHALLENGE 0120 2 SMA ->
		Указать размер: 42 ->
		Нажать кнопку "Добавить в корзину"
	Результат: В body в span элементе с классом "minicraft-count" значение станет равным 1


3. Удаление  товара из корзины
	Шаги:
		Странциа https://lacoste.ru/catalog/polo-muzh/polo_lacoste_classic_fit_s_dlinnym_rukavom_color_031/ ->
		Выбрать цвет: Зелёный ->
		Указать размер: 48 ->
		Нажать на кнопку "Добавить в корзину" ->
		Страница https://lacoste.ru/cart/ ->
		Нажать на кнопку "Удалить товар"
	Результат: На странице появится уведомление "Ваша корзина пуста"


4. Использование промокода SNOPNOW10
	Шаги:
		Страница https://lacoste.ru/cart/ ->
		Ввести в поле промокод SNOPNOW10 ->
		Нажать на кнопкпу "Применить" на той же панели ->
	Результат: Без промокода цена 4 482 рублей, а с ним стала 4033,8 рублей, изменения происходят в li с классом "cart-price-total"


5. Фильтраиция по полу
	Шаги: 
		Страница https://lacoste.ru/catalog/parfyum/ ->
		Выбрать пол: Мужской 
	Результат: На странице появится товар для мужчин. В свойтве filter будет стоять значение "Мужчины"


6. Проверка функции "Примерить онлайн"
	Шаги:
		Страница https://lacoste.ru/catalog/bryuki-muzhchiny/dzhinsy_lacoste_106_color_53A/ ->
		Нажать кнопку "Примерить онлайн" ->
		Указать размеры груди: 92 ->
		Указать размеры талии: 115 ->
		Указать рост: 176 ->
		Указать размеры бёдер: 75
	Результат: Будет выведено сообщение "Данный размер не подходит". Эти изменения будут выводится в div с классом info


7. Оформление заказа товара
	Шаги:
		Страница https://lacoste.ru/catalog/polo/ ->
		Выбираем товар Поло Lacoste ->
		Нажимаем кнопку "Добавить в корзину" ->
		Страница https://lacoste.ru/catalog/novye-postupleniya-obuv-zhenshchiny/carnaby_evo_118_6_spw_color_216/ ->
		Нажимаем кнопку "Добавить в корзину" ->
		Странциа https://lacoste.ru/cart/ ->
		Нажимаем кнопку оформить заказ 
	Результат: В теге p  c  классом "сart-empty-text" прописано   "Номер заказа: 236807",   " Сумма заказа: 13 698 руб"

8. Поиск товара по поисковой строке
	Шаги:
		Страница https://lacoste.ru/->
		Вводим текст в поисковую строку "Поло"
		Нажимаем клавишу "Enter" 
		
	Результат: В div  с классом page-container появится список ul c классом search-result


9. Заказ нескольких товаров одного вида
	Шаги:
		Страница https://lacoste.ru/catalog/polo/ ->
		Добавить поло в корзину ->
		Страница https://lacoste.ru/cart/ ->
		Выбрать количество 3
	Результат: В select свойство option имеет значение 3


10. Заказ товара в 1 клик
	Шаги:
		Страница https://lacoste.ru/catalog/chasy-akses-zhenshchiny/chasy_lacoste_8_color_000/->
		Нажать кнопку "Купить в 1 клик"->
		Ввести данные ->
		Нажать кнопку "Отправить" 
	Результат: Выведет сообщение "Заказ оформлен"
